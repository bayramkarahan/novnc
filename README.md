# novncservice eklentisi
novncservice x11vnc kurulu ve başlatılmış ise tarayıcı üzerinden erişim sağlar.
<p> Bu paket 5905 portunu 6085 üzerinden paylaştırıyor.</p>
Tarayıcıya;

` 
ip:6085/vnc.html 
`

adressinden masaüstüne erişilebilir.
